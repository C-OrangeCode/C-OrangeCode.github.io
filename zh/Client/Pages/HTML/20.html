                                                            提示

                                                            当在 HTML 中设置颜色时，通常会使用颜色值来指定元素的颜色。在 HTML 和 CSS 中，有多种方式来表示颜色，包括颜色名称、十六进制值、RGB 值和 HSL 值。

                                                            • 如需复制颜色，鼠标左键 L mouse 双击选中颜色代码 ctrl+c 复制
                                                            • 如果颜色有色差请调整为深色主题查看

                                                            下面是对这些表示颜色的方式的详细解释：

                                                            # 颜色值

                                                            # 十六进制

                                                            颜色可以用十六进制表示，通常是以 # 开头，后面跟着 3 或 6 位十六进制数字，分别代表红色(R)、绿色(G)和蓝色(B)的分量，有以下两种写法。

                                                            • 三位十六进制表示法：

                                                              • 使用三位十六进制数来表示颜色，每个分量由一个十六进制数字表示。通常，三位十六进制表示法是对六位表示法的简写，例如 #F00 表示红色，#0F0 表示绿色，#00F 表示蓝色。这种表示法在某些情况下可以节省空间，但无法表示所有的颜色。
                                                            示例
                                                            <div class="text-demo" style="color:#F00 ">三位十六进制红色文本</div>
<div class="text-demo" style="color:#0F0 ">三位十六进制绿色文本</div>
<div class="text-demo" style="color:#00F ">三位十六进制蓝色文本</div>

                                                            div {
  float: left;
  margin-left: 10px;
}
.text-demo {
  text-align: center;
  font-weight: bolder;
}

                                                            • 六位十六进制表示法：

                                                              • 使用六位十六进制数来表示颜色，每个分量由两个十六进制数字表示。例如，#FF0000 表示红色，#00FF00 表示绿色，#0000FF 表示蓝色。这种表示法可以表示更多种颜色，而且更直观和通用。
                                                            示例
                                                            <div class="text-demo" style="color:#FF0000 ">六位十六进制红色文本</div>
<div class="text-demo" style="color:#00FF00 ">六位十六进制绿色文本</div>
<div class="text-demo" style="color:#0000FF ">六位十六进制蓝色文本</div>

                                                            div {
  float: left;
  margin-left: 10px;
}
.text-demo {
  text-align: center;
  font-weight: bolder;
}

                                                            # 带 alpha 通道的十六进制

                                                            除了普通的六位十六进制表示颜色的方式外，还可以使用带 alpha 通道的八位十六进制值来表示颜色，其中前两位表示红色分量、中间两位表示绿色分量、后两位表示蓝色分量，最后两位表示透明度。例如，#00000000 表示完全透明的黑色，#FF000080 表示半透明的红色。

                                                            示例

    红色半透明背景
  </div>
  <div class="text-demo" style="background-color:#00FF0080 ">
    绿色半透明背景
  </div>
  <div class="text-demo" style="background-color:#0000FF80 ">
    蓝色半透明背景
  </div>
</div>

                                                            # RGB

                                                            颜色也可以使用 RGB 值来表示，RGB 分别代表红色（Red）、绿色（Green）和蓝色（Blue）的分量。RGB 值的表示方式是 rgb(红色值, 绿色值, 蓝色值)，每个分量的值范围是 0 到 255。例如，rgb(255, 0, 0) 表示纯红色，rgb(0, 255, 0) 表示纯绿色，rgb(0, 0, 255) 表示纯蓝色。

                                                            示例

<div class="text-demo" style="color:rgb(0, 255, 0) ">RGB绿色文本</div>
<div class="text-demo" style="color:rgb(0, 0, 255) ">RGB蓝色文本</div>

                                                            # RGBA

                                                            RGBA 是 RGB 的扩展，多了一个表示透明度的分量（Alpha）。RGBA 值的表示方式是 rgba(红色值, 绿色值, 蓝色值, 透明度)，其中透明度的取值范围是 0 到 1，0 表示完全透明，1 表示完全不透明。例如，rgba(255, 0, 0, 0.5) 表示半透明的红色。

  <div class="text-demo" style="background-color:rgba(0, 0, 255, 0.5) ">
    RGBA蓝色半透明背景

                                                            # HSL

                                                            • HSL 是一种基于颜色的色相（Hue）、饱和度（Saturation）和亮度（Lightness）来表示颜色的方式，写法为：hsl(色相, 饱和度, 亮度)。
                                                              • 色相的取值范围是：0-360，分别对应色相值：
                                                                • 0-60：红色到黄色
                                                                • 61-120：黄色到绿色
                                                                • 121-180：绿色到青色
                                                                • 181-240：青色到蓝色
                                                                • 241-300：蓝色到紫色
                                                                • 301-360：紫色到红色
                                                              • 饱和度的取值范围是：0%-100%。
                                                              • 亮度的取值范围是 0%-100%

                                                            这种方式可以更直观地描述颜色的特性，比如色调、饱和度和亮度。

                                                            示例
                                                            <div class="text-demo" style="color:hsl(0, 100%, 50%)">HSL红色文本</div>
<div class="text-demo" style="color:hsl(120, 100%, 50%)">HSL绿色文本</div>
<div class="text-demo" style="color:hsl(240, 100%, 50%)">HSL蓝色文本</div>

                                                            # HSLA

                                                            HSLA 是在 HSL 的基础上加了一个 透明度（Alpha） 它的取值范围是 0-1，1 表示完全不透明，0 表示完全透明,写法为：hsla(色相, 饱和度, 亮度，透明度)。

                                                            示例

    HSLA红色半透明背景

    HSLA绿色半透明背景
  </div>
  <div class="text-demo" style="background-color:hsla(240, 100%, 50%, 0.5)">
    HSLA蓝色半透明背景
