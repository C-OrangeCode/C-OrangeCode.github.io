                                                  # ASCII 字符集

                                                  ASCII ，它的全称是"美国信息交换标准代码"。它设计于 60 年代早期，是计算机和诸如打印机、磁带驱动器之类的硬件设备的标准字符集。
                                                  ASCII 是 7 比特字符集，包含了 128 个不同的字符值。
                                                  ASCII 支持 0-9 的数字，A-Z 大写和小写英文字母，以及一些特殊字符。
                                                  被广泛使用于现代计算机、HTML 和因特网上的字符集都是基于 ASCII 。
                                                  以下表格列举了 128 个 ASCII 字符以及对应的 HTML 实体编码。
